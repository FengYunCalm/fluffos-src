# Security Policy

## Reporting a Vulnerability

This fork does not publish a direct security contact channel. If you believe you have found a security vulnerability, use the platform’s built‑in security reporting tools or follow the upstream FluffOS security policy.

Please include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any known mitigations

We will review reports as soon as possible.

## Supported Versions

This fork tracks the `master` branch. Security fixes are applied to the latest code only.
